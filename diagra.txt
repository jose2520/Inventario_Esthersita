================================================================================
                   FLUJO DE DATOS DETALLADO (ESTRUCTURA)
================================================================================

   ENTRADA (Inputs)            PROCESAMIENTO (RAM)           SALIDA (Persistencia)
  ------------------          ---------------------         ----------------------
  > Nombre (str)    -----\    [ { "nombre": "X", ]    /--->  JSON (Estructurado)
  > Precio (float)  ------>     "precio": 0.0,    |          "Sync Automática"
  > Cantidad (int)  -----/      "cantidad": 0 }  ]    \--->  CSV (Hoja Cálculo)
                                                              "Export/Import"

================================================================================
                    ESPECIFICACIONES TÉCNICAS (Stack)
================================================================================

  * MOTOR:        Python 3.x
  * INTERFAZ:     ANSI Escaped CLI (Colores y Tablas ASCII)
  * PERSISTENCIA: Serialización JSON (UTF-8)
  * INTEGRACIÓN:  CSV Engine (DictWriter)
  * SEGURIDAD:    Validación de tipos (Try-Except) y Persistencia Atómica

--------------------------------------------------------------------------------
   "El código está diseñado bajo el principio de Responsabilidad Única (SRP)"
